# Quiz

Apenas um projeto de quiz simples.

## Como clonar

1. Clone o repositório:
   ```bash
   https://github.com/MonoV1/quiz.git
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Execute o script:
   ```bash
   python main.py
   ```

## Requisitos

- Python 3.x
- Acesso ao roteador (IP, usuário e senha)
